# APT Tests

This directory is reserved for isolated APT transaction tests.

Current automated coverage lives in:

- `scripts/test-local-repo.sh`
- `scripts/apt-signed-smoke-test.sh`
- `scripts/apt-pages-smoke-test.sh`
- `scripts/validate-apt-safety.sh`
- `scripts/build-and-validate.sh`

These tests must pass before publishing packages or starting any ISO release work.
